ClimaView

About the Project

ClimaView is a real-time weather app that uses the OpenWeather API to fetch and display accurate weather data for any location. Users can search for the current weather and forecast based on their city or GPS location.

Key Features

  • Real-time weather data
  • Search by city or GPS location
  • Forecast for up to 7 days
  • Minimalist and responsive design
