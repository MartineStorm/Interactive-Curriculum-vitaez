BuzzFinder

About the Project

BuzzFinder is a web app designed to help users find local events based on their interests. It aggregates events from multiple sources and allows users to search by category, location, and date.

Key Features

  • Event search by category and location
  • Real-time event updates
  • User-friendly and responsive design
  • Personalized event recommendations
