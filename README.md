Project Overview
This project is a modern, interactive CV designed to present my qualifications and skills dynamically. It includes:

Animated progress bars to showcase skills.
Percentage indicators within progress bars.
Buttons for downloading, printing, and visiting my GitHub profile.
Features
Interactive Progress Bars: Displays skill levels with smooth animations.
Download CV: Option to download the CV as a PDF.
Print CV: Option to print the CV directly.
GitHub Profile Button: Quick access to my GitHub profile.
Technologies Used
HTML: For the structure of the CV.
CSS: For styling and layout.
JavaScript: For interactive elements and functionality.
html2pdf.js: For generating a PDF version of the CV.

Usage
View the CV: Open the index.html file in your browser to see the interactive CV.
Download CV: Click the "Download CV" button to save a PDF version.
Print CV: Click the "Print CV" button to print the CV.
Visit GitHub Profile: Click the "Github Profile" button to view my GitHub profile.
